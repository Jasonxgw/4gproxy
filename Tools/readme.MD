##clientUtilities
You can use clientUtilities to do IP reset for Huawei E3372, the valid command formt is

```
clientUtilities -deviceMode e3372 -params $IP
```

For e8372 is:
```
clientUtilities -deviceMode e8372 -params $IP#user#password
```
